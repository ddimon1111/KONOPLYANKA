# WeedSimulator v2 (Discord.py + PyQt5)

Этот проект — экономический Discord-бот с GUI-запуском на PyQt5.

## Быстрый старт

1. Установите Python 3.14 (или совместимую версию) и зависимости:
   ```bash
   pip install -U discord.py PyQt5
   ```
2. Запустите GUI:
   ```bash
   python weedsimulator_v2.py
   ```
3. Вставьте токен, введите **Guild ID** сервера и нажмите **Start**.

---

## Тутор: как настроить бота в Discord Developer Portal

### 1) Создать приложение
1. Откройте: https://discord.com/developers/applications
2. Нажмите **New Application**.
3. Задайте имя (например, `WeedSimulator v2`) и создайте приложение.

### 2) Создать бота
1. Откройте вкладку **Bot** в вашем приложении.
2. Нажмите **Add Bot**.
3. Включите опции (если нужны):
   - **Presence Intent**
   - **Server Members Intent**
   - **Message Content Intent** (обязательно для `!`-команд)

### 3) Получить токен
1. Во вкладке **Bot** нажмите **Reset Token** (или **Copy** если уже есть токен).
2. Скопируйте токен и вставьте его в GUI приложения.

> ⚠️ Никому не передавайте токен. Если утёк — сразу делайте **Reset Token**.

### 4) Получить Guild ID
1. В Discord откройте **User Settings → Advanced**.
2. Включите **Developer Mode**.
3. ПКМ по серверу → **Copy Server ID**.
4. Вставьте ID в поле **Guild ID** в GUI.

### 5) Пригласить бота на сервер
#### Вариант A (через кнопку в GUI)
- Введите Guild ID и нажмите **Copy Invite Link**.
- Откройте ссылку в браузере и подтвердите добавление бота.

#### Вариант B (вручную)
Соберите ссылку формата:
```
https://discord.com/oauth2/authorize?client_id=YOUR_CLIENT_ID&permissions=8&scope=bot%20applications.commands&guild_id=YOUR_GUILD_ID
```

- `YOUR_CLIENT_ID` — ID вашего приложения (General Information → Application ID)
- `YOUR_GUILD_ID` — ID вашего сервера

### 6) Проверить запуск
1. Нажмите **Start** в GUI.
2. Дождитесь статуса **Bot online ✅**.
3. В нужном сервере используйте `!help` или `!menu`.

## Частые проблемы

### `Error: Session is closed`
- Обычно появляется после неудачного старта/остановки или старого состояния.
- Перезапустите GUI и попробуйте снова.
- Убедитесь, что токен актуальный (при необходимости Reset Token).

### Бот не отвечает на команды
- Проверьте, что включён **Message Content Intent**.
- Проверьте, что введён корректный **Guild ID**.
- Убедитесь, что бот имеет права читать/писать сообщения.

### Бот не появляется онлайн
- Проверьте интернет и firewall.
- Убедитесь, что токен от нужного приложения.
- Посмотрите строку статуса в GUI (она показывает текст ошибки).

## Полезные команды
- `!menu` — кнопочное меню действий
- `!help` — список команд
- `!balance` — баланс игрока
- `!inventory` — инвентарь
- `!met` — обмен косяков в валюту мёт
- `!badges shop` — магазин бейджей
